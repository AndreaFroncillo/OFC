# Olimpia Club House Documentation

This directory contains the architectural and technical documentation of the Olimpia Club House project.

The documentation is organized by language and divided into architectural volumes, cross-cutting guidelines, and Architecture Decision Records.

---

## Available Languages

- [English Documentation](en/)
- [Documentazione italiana](it/)

The English documentation is the authoritative source.

The Italian documentation is maintained as a translation and may temporarily lag behind the corresponding English version while architectural documents are being revised.
